# Gemini Tales

Gemini Tales is an interactive AI storytelling companion for children.
Stories adapt in real time to each child's age, language, and interests,
powered by the Gemini API.

## Setup

```bash
pip install -r requirements.txt
export GEMINI_API_KEY=...
python -m app.main
```
